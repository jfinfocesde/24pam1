# Semana 10

## PostgreSQL: Un Sistema de Gestión de Bases de Datos Relacionales (RDBMS)

[https://www.postgresql.org/](https://www.postgresql.org/)

PostgreSQL es un **sistema de gestión de bases de datos relacional (RDBMS)** de código abierto, conocido por su **fiabilidad, integridad de datos y cumplimiento de los estándares SQL**. Es una herramienta poderosa y versátil, que se usa en una amplia gama de aplicaciones, desde sitios web pequeños hasta grandes empresas.

**Características principales:**

* **Cumplimiento de los estándares SQL:** PostgreSQL implementa casi por completo el estándar SQL, lo que lo hace compatible con una amplia gama de herramientas y aplicaciones.
* **Fiabilidad y seguridad:**  Contiene características de recuperación de errores y seguridad robustas, lo que garantiza la integridad de los datos y la protección contra accesos no autorizados.
* **Transacciones ACID:** PostgreSQL garantiza que las transacciones sean atómicas, consistentes, aisladas y duraderas, lo que garantiza la integridad de los datos en entornos de alta concurrencia.
* **Funcionalidad avanzada:** Ofrece características como funciones y tipos de datos definidos por el usuario, triggers, vistas, procedimientos almacenados y herencia de tablas.
* **Compatibilidad multiplataforma:** Se puede ejecutar en una variedad de sistemas operativos, incluidos Linux, macOS, Windows y Unix.
* **Comunidad activa y soporte:** Tiene una gran comunidad activa de usuarios y desarrolladores, lo que proporciona un soporte sólido y una amplia gama de recursos.

**Beneficios de usar PostgreSQL:**

* **Costo-efectivo:** PostgreSQL es de código abierto, por lo que no hay costos de licencia.
* **Flexible y escalable:** Se puede adaptar a diferentes necesidades y tamaños de aplicaciones.
* **Sólido y confiable:** Su sólida arquitectura garantiza la integridad de los datos y el rendimiento.
* **Seguridad mejorada:** Ofrece funciones de seguridad robustas para proteger los datos.

**Usos comunes de PostgreSQL:**

* **Sitios web y aplicaciones web:** PostgreSQL es una opción popular para bases de datos de sitios web y aplicaciones web.
* **Aplicaciones empresariales:** Se utiliza para gestionar datos críticos en aplicaciones empresariales, como gestión de clientes, finanzas y logística.
* **Análisis de datos:** Se utiliza para almacenar y analizar grandes conjuntos de datos, debido a su capacidad de manejar tablas grandes y complejas.
* **Desarrollo de software:** Es una herramienta popular para desarrolladores de software que necesitan un sistema de gestión de bases de datos confiable y eficiente.

## PostgreSQL on Render: Una Solución Completa y Fácil para tu Base de Datos

[https://docs.render.com/databases](https://docs.render.com/databases)

**PostgreSQL on Render** es una forma **rápida, fácil y escalable** de ejecutar tu base de datos PostgreSQL en la nube, sin necesidad de preocuparte por la configuración, el mantenimiento o la gestión de servidores. Render es una plataforma de desarrollo que ofrece una experiencia de "servidorless" (sin servidor) para aplicaciones web y bases de datos.

**Beneficios de utilizar PostgreSQL on Render:**

* **Instalación y configuración sencillas:** No necesitas configurar ningún servidor o instalar software. Simplemente selecciona tu plan y Render se encarga de todo.
* **Escalabilidad automática:** Render ajusta automáticamente los recursos de tu base de datos según la demanda, asegurando que siempre tengas el rendimiento que necesitas.
* **Integración sin problemas con otras herramientas:** Render se integra fácilmente con otras herramientas y servicios de desarrollo, como Git, Docker y CI/CD.
* **Seguridad y redundancia:** Render ofrece una arquitectura robusta con múltiples niveles de seguridad y redundancia, lo que garantiza la disponibilidad de tus datos.
* **Precios transparentes y flexibles:** Render ofrece planes flexibles y transparentes que se ajustan a tu presupuesto y necesidades.

**Cómo funciona PostgreSQL on Render:**

1. **Crea una cuenta en Render:** El proceso es rápido y sencillo.
2. **Crea un servicio PostgreSQL:** Selecciona el plan que mejor se adapte a tus necesidades y especifica el nombre de tu base de datos y sus credenciales.
3. **Conecta tu aplicación:** Puedes conectar tu aplicación web a la base de datos utilizando las credenciales proporcionadas por Render.
4. **Empieza a usar:** Ahora puedes empezar a utilizar tu base de datos PostgreSQL desde tu aplicación, sin necesidad de preocuparte por la gestión del servidor.


## DBeaver: Tu Herramienta Universal para la Gestión de Bases de Datos

[https://dbeaver.io/](https://dbeaver.io/)

DBeaver es una **herramienta de gestión de bases de datos gratuita y de código abierto** que ofrece una interfaz gráfica para interactuar con una amplia variedad de sistemas de gestión de bases de datos (DBMS). Es una herramienta poderosa y versátil que puede utilizarse para:

**Características principales:**

* **Soporte multiplataforma:** Disponible para Windows, macOS y Linux.
* **Conectividad a múltiples DBMS:** Soporta más de 40 DBMS, incluyendo PostgreSQL, MySQL, Oracle, SQL Server, MongoDB, Cassandra, etc.
* **Interfaz intuitiva:** Ofrece una interfaz gráfica amigable con funciones de arrastrar y soltar, autocompletado de código y herramientas de edición de datos.
* **Funcionalidades avanzadas:**  Incluye funciones como el editor de SQL, la depuración de consultas, la generación de informes, la creación de diagramas de bases de datos, la sincronización de esquemas y la migración de datos.
* **Extensible mediante plugins:** DBeaver se puede extender con plugins para agregar funciones adicionales y soporte para más DBMS.
* **Seguridad y privacidad:** Permite la encriptación de conexiones y el acceso a datos sensibles con permisos.
* **Compatibilidad con diferentes lenguajes:** Ofrece soporte para SQL, PL/SQL, T-SQL, etc.

**Beneficios de usar DBeaver:**

* **Gratis y de código abierto:** No hay costos de licencia y puedes contribuir al desarrollo de la herramienta.
* **Versátil y flexible:** Se adapta a una amplia gama de necesidades y tareas de administración de bases de datos.
* **Fácil de usar:** Su interfaz intuitiva la hace accesible para usuarios de todos los niveles de experiencia.
* **Productividad mejorada:** Las funciones avanzadas optimizan el flujo de trabajo y mejoran la productividad.
* **Amplia comunidad de usuarios:** DBeaver cuenta con una gran comunidad de usuarios que ofrece soporte y recursos adicionales.

**Aplicaciones comunes de DBeaver:**

* **Desarrolladores:** Para conectarse a bases de datos, ejecutar consultas, crear y editar tablas y datos.
* **Administradores de bases de datos:** Para gestionar y administrar usuarios, permisos, esquemas y datos.
* **Analistas de datos:** Para explorar y analizar datos, crear informes y realizar consultas complejas.
* **Estudiantes e investigadores:** Para aprender y practicar la administración de bases de datos.

### 3. Consultas para Operar sobre la Tabla "productos":

1. **Mostrar todos los productos:**

```sql
SELECT * FROM productos;
```

2. **Mostrar los productos con precio mayor a $50:**

```sql
SELECT * FROM productos WHERE precio > 50;
```

3. **Mostrar los productos de la categoría "Ropa" ordenados por precio:**

```sql
SELECT * FROM productos WHERE categoria = 'Ropa' ORDER BY precio ASC;
```

4. **Contar el número de productos por categoría:**

```sql
SELECT categoria, COUNT(*) FROM productos GROUP BY categoria;
```

5. **Mostrar los productos con stock menor a 10:**

```sql
SELECT * FROM productos WHERE stock < 10;
```

6. **Actualizar el precio de un producto:**

```sql
UPDATE productos SET precio = 29.99 WHERE id = 1;
```

7. **Eliminar un producto:**

```sql
DELETE FROM productos WHERE id = 3;
```

8. **Mostrar los productos con un precio entre $10 y $30:**

```sql
SELECT * FROM productos WHERE precio BETWEEN 10 AND 30;
```

9. **Mostrar los productos que fueron creados en marzo de 2023:**

```sql
SELECT * FROM productos WHERE fecha_creacion BETWEEN '2023-03-01' AND '2023-03-31';
```

10. **Añadir una nueva columna a la tabla:**

```sql
ALTER TABLE productos ADD COLUMN descripcion VARCHAR(255);
```
